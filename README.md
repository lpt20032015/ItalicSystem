# Italic System
This registry tweak will make your system use the italic system font just like in the Windows PE/Recovery Menu.

## Versions it works on
- Windows 7
- Windows 8
- Windows 8.1
- Windows 10
- Windows 11

## Important!

On newer Windows 10/11 builds, some UI parts may ignore the registry override because Microsoft hardcodes fonts in certain modern apps.
Some text can become visually broken or cramped because Segoe UI Italic was not designed as a full system UI replacement.
Windows updates can sometimes reset the font mappings.
The files require administrator privileges because they edit HKEY_LOCAL_MACHINE.
